# TBT

Unity turn-based tactics prototype built around grid movement, action selection, and tactical combat interactions.

## Project Highlights

- Grid-based movement and range visualization
- Turn system for player and enemy phases
- Unit action architecture for move, shoot, melee, grenade, spin, and interact actions
- A* pathfinding with walkability updates
- Enemy AI action evaluation
- Camera controls, screen shake, unit selection, and in-world/UI feedback
- Destructible crates, doors, projectiles, ragdoll spawning, and health handling

## Tech

- Unity 6000.3.3f1
- C#
- Universal Render Pipeline
- Unity Input System
- Cinemachine

## Repository Notes

This public portfolio repository excludes third-party Asset Store packs, source animation files, and source audio files to avoid redistributing licensed assets. The gameplay code, Unity project settings, packages, scenes, prefabs, materials, and project structure are kept for review.

To open locally, use Unity Hub with Unity `6000.3.3f1` or a compatible Unity 6 editor version, then restore any missing external visual, animation, or audio assets if you need the original scene presentation.
